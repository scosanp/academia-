# AI Start: ChatGPT и промпты с нуля

Интерактивный статический сайт-курс на русском языке для новичков, которые хотят системно пользоваться искусственным интеллектом, ChatGPT и промпт-инжинирингом.

## Что реализовано

- Landing page с hero-блоком, ценностью курса, структурой, CTA и описанием формата обучения.
- Страница курса с 10 модулями, 30 уроками, статусами «Доступен», «Пройден», «Заблокирован».
- Страница урока с теорией, примерами промптов, мини-практикой и главной мыслью.
- Тесты 1 из 4 с мгновенной обратной связью, подсветкой верного и неверного ответа.
- Линейное прохождение: следующий урок открывается после теста на 70% или выше.
- Прогресс в `localStorage`, повторное прохождение тестов и сброс прогресса.
- Информационная база из 42 карточек с поиском и фильтрами по категориям.
- Страница прогресса, FAQ и 404.
- GitHub Pages workflow для статического деплоя.

## Стек

- Vite + React + TypeScript
- React Router с `HashRouter` для совместимости с GitHub Pages
- Tailwind CSS + глобальные design tokens
- Vitest + React Testing Library
- ESLint + Prettier

## Локальный запуск

```bash
npm install
npm run dev
```

Откройте URL, который покажет Vite, обычно `http://localhost:5173`.

## Проверки

```bash
npm run lint
npm test
npm run build
```

## Деплой на GitHub Pages

1. Откройте настройки репозитория на GitHub.
2. Перейдите в `Settings` → `Pages`.
3. В `Build and deployment` выберите `Source: GitHub Actions`.
4. Запушьте ветку с workflow `.github/workflows/pages.yml`.
5. После merge в `main` workflow соберет `dist` и опубликует сайт.

## Структура данных курса

- `src/data/course.ts` — модули, уроки, примеры промптов, практика и тесты.
- `src/data/knowledgeBase.ts` — категории и карточки информационной базы.
- `src/lib/quiz.ts` — подсчет результата теста и состояния вариантов ответа.
- `src/lib/progress.ts` — проходной балл, статусы уроков, статистика курса.
- `src/lib/storage.ts` — безопасная работа с `localStorage`.
- `src/hooks/useProgress.ts` — React-хук для прогресса и сохранения попыток.

- Запустить `npm run lint`.

- Запустить `npm run build`.
