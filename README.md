# РезюмеПрофи MVP

Статический бесплатный прототип сервиса для создания резюме с быстрым мастером, чат-интервью, загрузкой старого резюме, предпросмотром, экспортом, заявками на ручную проверку и личное сопровождение.

## Локальный запуск

```bash
node server.mjs
```

Откройте `http://localhost:4173`.

Если на компьютере установлен npm, можно также запустить:

```bash
npm run dev
```

## Публикация

Проект готов для GitHub Pages. Workflow в `.github/workflows/pages.yml` публикует содержимое репозитория после push в ветку `main`.

Ожидаемая публичная ссылка:

```text
https://scosanp.github.io/rezume/
```

## Что работает в прототипе

- главная страница с быстрым стартом;
- быстрый мастер резюме;
- подробное чат-интервью;
- загрузка TXT/PDF/DOCX с mock-извлечением данных;
- подтверждение извлеченных данных;
- mock AI-генерация резюме без выдумывания фактов;
- предпросмотр и редактирование;
- 4 шаблона: classic, executive, production, modern;
- экспорт PDF, DOCX и TXT;
- заявки на ручную проверку;
- заявка на личное сопровождение;
- локальная админ-панель заявок;
- страницы Privacy, Terms, Consent;
- хранение данных в `localStorage`.
